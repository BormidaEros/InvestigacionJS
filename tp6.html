1°

En el mercado actualemnte encontramos las siguientes versiones

  • Javascript 1: Fue la primera version, nacio con el Netscape 2.0, soportaba multiples instruccione y funciones, casi todas las que existen ahora ya se intrudujieron en su primera vercion
  • Javascript 1.1: Es la version de javascript que se implemento para los navegadores 3.0, en esta vercion se implementan los arreglos.
  • Javascript 1.2: La version de los navegadores 4.0, era diferente para las plataformas de Microsoft y Netscape, ya que los navegadores se fueron desarrollando de diferentes maneras.
  • Javascript 1.3: Version que implementan los navegadores 5.0, se eliminan diferencias entre navegadores.
  • Javascript 1.5: Version actual, en el momento de escrivir estas lineas, que implementa Netscape 6.
  • JScript: Una version mas de javascript para navegadores de Microsoft.
  • ECMAscript 5: Una version de javascript que funciona en todos los navegadores por igual

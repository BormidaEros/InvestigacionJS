¿Qué versiones de JavaScript hay en el mercado?.

En el mercado actualemnte encontramos las siguientes versiones

  • Javascript 1: Fue la primera version, nacio con el Netscape 2.0, soportaba multiples instruccione y funciones, casi todas las que existen ahora ya se intrudujieron en su primera vercion
  • Javascript 1.1: Es la version de javascript que se implemento para los navegadores 3.0, en esta vercion se implementan los arreglos.
  • Javascript 1.2: La version de los navegadores 4.0, era diferente para las plataformas de Microsoft y Netscape, ya que los navegadores se fueron desarrollando de diferentes maneras.
  • Javascript 1.3: Version que implementan los navegadores 5.0, se eliminan diferencias entre navegadores.
  • Javascript 1.5: Version actual, en el momento de escrivir estas lineas, que implementa Netscape 6.
  • JScript: Una version mas de javascript para navegadores de Microsoft.
  • ECMAscript 5: Una version de javascript que funciona en todos los navegadores por igual

    ¿Cómo son todas las formas de declaración de funciones en JS?

    En javascript existen varios tipos de funciones principales

  • Funciones por declaracion:Es una funcion que existira a lo largo de todo nuestro codigo
  • Funciones por expresion:Es un tipo de uncion que se guarda dentro de variables, para posterioremente ejecutar las variables.
  • Funciones Como Objetos:En Javascript se pueden declarar funciones en forma de objetos es importante saberlo ya que en javascript todo se puede declarar como objetos
  • Funciones anonimas o Funciones lamba:son un tipo de funciones que se declaran sin nombre, se alojan en en una variable y haciendo referencia a a ella cada vez que queramos utilizarla.
  • Callbacks:Son una especie de funciones anindadas, podemos utilizar como parametro una funcion ya creada.

    • ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?

    El objeto dom es una interfaz de programacion que permite acceder al codigo para modificar su estructura, contenido y estilo. El modelo DOM define la estructura lógica de los documentos y el modo en que se accede a ellos y se los manipula.

    ¿Qué función tiene en JS el Object Map?

    El objet map crea un nuevo arreglo con los resultados de una función indicada, aplicados a cada uno de sus elementos.
